# qlip-serve-common

Common abstractions and utilities for qlip.serve.* libraries

## Installation

```bash
pip install qlip-serve-common
```

## Requirements

- Python ≥ 3.10
- numpy < 2.0.0 (required for tritonserver compatibility)
- protobuf ≥ 5.26.0
- pillow ≥ 10.2.0

About Upstream Sync

Upstream sync allows you to push changes from your local Plex collections back to the original collages on RED/OPS sites.

  • Requirements: You must own the collages on the source site
  • Safety: Only albums that are missing from the upstream collage will be added
  • Selection: You can choose exactly which albums to sync for each collage
  • Rate limits: The sync process respects API rate limits automatically
